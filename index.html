Sobre Mim

Sou estudante do 4º período de Ciência da Computação na Newton Paiva, apaixonado por tecnologia e desenvolvimento de software. Com um grande interesse em resolver problemas e criar soluções eficientes, estou em busca do meu primeiro emprego na área de TI, onde posso aplicar os conhecimentos adquiridos ao longo do curso e continuar aprendendo. Tenho uma verdadeira paixão por videogames, sempre buscando entender os aspectos técnicos que tornam os jogos possíveis. Além disso, sou entusiasta de basquete e automobilismo, áreas que me inspiram pela dinâmica, estratégia e inovação. Acredito que meu amor por essas áreas reflete meu interesse por desafios e por trabalhar com tecnologia de ponta. Estou em constante busca por crescimento profissional, sempre me atualizando sobre as últimas tendências e práticas do setor de tecnologia. Tenho um forte desejo de contribuir em um ambiente de trabalho colaborativo e, ao mesmo tempo, de aprimorar minhas habilidades técnicas e criativas.

🌟 Aplicação Web para Gerenciamento de Tarefas

Este projeto é uma aplicação web de gerenciamento de tarefas, projetada para facilitar o controle das atividades diárias e aumentar a produtividade dos usuários. Com uma interface intuitiva e funcionalidades eficientes, a aplicação permite adicionar, editar e excluir tarefas de forma prática.

🔹 Tecnologias Utilizadas:

  • ✅ HTML5, CSS3 e JavaScript – Para estruturação, estilo e interatividade no front-end.
  • ✅ Node.js e Express – Para o desenvolvimento do back-end e criação de APIs.
  • ✅ MongoDB – Banco de dados NoSQL para armazenar as tarefas de forma eficiente.
  • ✅ Bootstrap – Framework para garantir um design responsivo e moderno.

🔥 Funcionalidades:

  • ✔️ Criação, edição e exclusão de tarefas.
  • ✔️ Visualização das tarefas organizadas por prioridade e data.
  • ✔️ Marcação de tarefas como concluídas.
  • ✔️ Armazenamento de dados de forma segura no banco de dados.
  • ✔️ Interface responsiva e fácil de usar em dispositivos móveis e desktop.

🚀 Ideal para quem busca aumentar a produtividade pessoal e profissional, com um design simples e intuitivo.

📦 Sistema de Controle de Estoque

Este projeto consiste em um sistema de controle de estoque, projetado para facilitar o gerenciamento de produtos, entradas e saídas de mercadorias. O sistema permite o cadastro de novos produtos, controle das quantidades em estoque e a geração de relatórios detalhados, proporcionando uma visão clara e eficiente sobre o estado do estoque.

🔹 Tecnologias Utilizadas:

  • ✅ HTML, CSS e JavaScript – Para a criação de uma interface web responsiva e interativa.
  • ✅ Node.js e Express – Para o desenvolvimento do back-end e criação de APIs para manipulação de dados.
  • ✅ MySQL – Banco de dados relacional para armazenar informações de produtos, entradas e saídas.
  • ✅ Autenticação de usuários – Sistema de login para controle de acessos e permissões.

🔥 Funcionalidades:

  • ✔️ Cadastro de novos produtos com informações detalhadas.
  • ✔️ Controle de entradas e saídas de mercadorias.
  • ✔️ Geração de relatórios sobre estoque atual e produtos em falta.
  • ✔️ Pesquisa e filtros para facilitar a busca por produtos.
  • ✔️ Sistema de login com diferentes permissões para usuários (administrador, vendedor, etc.).

🚀 Ideal para empresas que precisam de um controle eficiente do seu estoque, com uma interface simples e funcional.
